Process

This page describes the process in creating this website

Part I: Information Architecture & Navigation chart

The basic information architecture chart includes:

  • Homepage
  • Subsections
  • Contents under subsections
  • Footer

The chart is formatted in a hierarchical structure that shows the relationship between blocks

The navigation chart is a next step of planning:
There are three types of navigations in this site - [1].Structural, [2].Associative, and [3].Utility

Color is used to differentiate the type of navigation

  • Structural Navigation: Green Line
  • Global Navigation: Orange Blocks
  • Associative navigation: Orange Lines
  • Contextual Navigation: Blue Blocks
  • Utility Navigation: Green Blocks

LOGO

The background color of the logo matches the color on the navigation panel

Following up with description of my Interests and Educations

Users can click the buttons under Interests & Educations to travel to other pages (Gallery & resume)

Resume page

Part III: Accessibility considerations

Proper use of headers

h1, h2, h3 are used properly according to importance of content

E.g. h1 is used in jumbotron; h2 is used as title to separate contents

Maintaining a similar structure for all pages

Avoid confusion when users are traveling between pages

E.g. present of jumbotron on every pages indicating which page they are on

Identified languages of the HTML document

The follow code is present in all HTML files:

"html lang="en""

Use of languages

Languages used in pages does not require a high reading level

E.g. Avoided unusual/technical terms, alt is present for all links and images

Work on all screens

Content of the site can be displayed in any screen size

Content resizes automatically

Proper use of color/color contrast

Ensure that readers can easily read the text on this site

E.g. Contrast Ratio of 18.54:1 for text on navigation panel
